# mini Ryudo
Repositório de firmware genérico do **mini Ryudo** (Mini Robot-Sumo - 500g - Radio Controlled). Possui implementação do receptor paralelo, com leitura de canais via PPM.

## Microcontrolador
STM32F103C8T6 (Bluepill)
